GitHub

Passo a Passo para a administração de um repositório

O que é o GitHub?

GitHub é uma plataforma de hospedagem de código-fonte com controle de versão usando o Git. Ele permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos privados e/ou Open Source de qualquer lugar do mundo.

Criando sua conta

  1. Para criar sua conta, acesse GitHub.com
  2. No canto superior direito, clique em Sign Up
  3. Insira seus dados e clique em Create an Account
  4. Depois, selecione o seu plano GitHub e clique em Continue
  5. Informe sua experiência com programação e clique em Submit

Criando seu Repositório

  1. Acesse GitHub.Com
  2. No canto superior direito, clique em + e então, em New repository.
  3. Nomeie Seu Repositório.
  4. Escreva uma descrição curta.
  5. Selecione Initialize this repository with a README.
  6. Selecione Create Repository.

Baixando o GIT

  1. Efetue o Download do GIT, acessando GitHub Download
  2. Após a instalação, o programa estará disponível na Área de Trabalho.

Clonando seu Repositório

  1. Clique com o botão direito do mouse na Área de Trabalho e clique em GIT GUI here
  2. Selecione Clone Existing Repository
  3. Em GitHub.com, selecione seu repositório e selecione Clone or Download
  4. Copie o endereço do repositório e cole-o em Source Location
  5. Em Target Directory, defina o nome da pasta onde o repositório será baixado

Salvando seus arquivos no Repositório

  1. Adicione os arquivos desejados na pasta onde o repósitório foi clonado.
  2. Depois, clique com o botão direito na pasta e selecione Git GUI here
  3. No programa, selecione Edit >> Options
  4. Na aba Global, insira seu nome de usuário e E-mail do GitHub e clique em Ok
  5. selecione Rescan para o escaneamento das mudanças feitas
  6. Clique em Stage Changed e em Sim
  7. Depois, insira um texto informando a modificação feita
  8. Clique em Commit
  9. Clique em Push e depois, em Push novamente
  10. Uma caixa será aberta, onde você deverá inserir seu Nome de Usuário e Senha
  11. Para verificar se os arquivos foram enviados para o GIT, acesse GitHub.com e selecione seu repositório
